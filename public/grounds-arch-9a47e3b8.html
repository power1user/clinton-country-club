System Architecture

Build · serve · sync

How a code push lands in a member's pocket. GitHub builds → Cloudflare Pages serves → Supabase backs everything. Realtime, Auth, Storage, Edge Functions all hang off the same project. Web Push fanout is self-hosted via VAPID.
